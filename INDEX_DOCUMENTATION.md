## 📥 TÉLÉCHARGEMENTS

### Documentation complète (PDF)

Pour générer un PDF de toute la documentation :

```bash
# Installer pandoc
sudo apt-get install pandoc

# Générer le PDF
pandoc README.md GUIDE_UTILISATION.md GUIDE_TECHNIQUE.md \
  CONFIGURATION_SCANNER.md DEPLOIEMENT.md INSTALLATION_EXPORTS.md \
  COMMANDES.md CHANGELOG.md RECAPITULATIF.md CONTACT.md \
  RESUME_EXECUTIF.md \
  -o documentation_complete_ong_made.pdf \
  --toc --toc-depth=3 \
  --pdf-engine=xelatex
```

### Documentation par thème

**Guide utilisateur** :
```bash
pandoc README.md GUIDE_UTILISATION.md RECAPITULATIF.md CONTACT.md \
```
